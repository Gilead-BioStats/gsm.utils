# Build template content for an example

Build template content for an example

## Usage

``` r
build_example_template(strName, strType, strDetails, intIndex)
```

## Arguments

- strName:

  Character. Display name of the example.

- strType:

  Character. Example type.

- strDetails:

  Character. Optional description.

- intIndex:

  Numeric. Optional ordering index.

## Value

Character vector of template lines.
