# Update pkgdown YAML with example menu entries

Update pkgdown YAML with example menu entries

## Usage

``` r
update_pkgdown_examples(pkgdown_yml, html_files, metadata)
```

## Arguments

- pkgdown_yml:

  Character. Path to `_pkgdown.yml`.

- html_files:

  Character vector of example HTML files.

- metadata:

  Data frame of example metadata.

## Value

`NULL` invisibly.
