# Add example menu items to pkgdown YAML

Add example menu items to pkgdown YAML

## Usage

``` r
add_pkgdown_examples_to_yaml(pkgdown_yaml, html_files, metadata)
```

## Arguments

- pkgdown_yaml:

  List representation of `_pkgdown.yml`.

- html_files:

  Character vector of example HTML files.

- metadata:

  Data frame of example metadata.

## Value

Updated pkgdown YAML list.
